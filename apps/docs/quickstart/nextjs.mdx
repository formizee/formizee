### Prerequisites
To get the most out of this guide, you'll need to:

* [<u>Create a Formizee account</u>](https://dashboard.formizee.com)
* [<u>Have a form created</u>](/guides/create-your-first-form)

## 1. Create a Next.js project

<Tabs>
  <Tab title="pnpm">
  ```bash
    pnpm create-next-app@latest
  ```
  </Tab>
  <Tab title="npm">
  ```bash
    npx create-next-app@latest
  ```
  </Tab>
  <Tab title="bun">
  ```bash
    bun create-next-app@latest
  ```
  </Tab>
  <Tab title="yarn">
  ```bash
    yarn create-next-app@latest
  ```
  </Tab>
</Tabs>

## 2. Update the main page
Update the file under `src/app/page.tsx` with the next code

```tsx page.tsx {1}
// Add here your Endpoint URL
const ENDPOINT_URL = "https://api.formizee.com/v1/f/enp_123456";

export default function Page() {
	return (
		<form method="post" action={ENDPOINT_URL}>
			<label htmlFor="name">
				Name
				<input id="name" autoComplete="name" name="name" />
			</label>
			<label htmlFor="email">
				Email
				<input id="email" autoComplete="email" name="email" />
			</label>
			<button>Submit</button>
		</form>
	);
}
```

## 3. Run in local

```bash
  pnpm run dev
```
